## Response Note API

| ----------| ----------------------------------  |
| POST      | `http://localhost:5000/notes     `  |
| GET       | `http://localhost:5000/notes     `  | 
| GET by ID | `http://localhost:5000/notes/noteID`|
| PUT/UPDATE| `http://localhost:5000/notes/noteID`|
